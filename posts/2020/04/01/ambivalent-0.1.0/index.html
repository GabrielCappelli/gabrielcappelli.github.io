Ambivalent 0.1.0

Wed 01 April 2020 ⋅ Gabriel Cappelli ⋅ 0 mins

This is Ambivalent's first release! Ambivalent is a lightweight theme for Pelican, with a responsive design and light/dark themes.

I plan to use Pelican and Ambivalent on my own blog and will be adding more features as I have a need for them.

Features

  • Lightweight
  • Responsive design
  • Light and Dark themes
  • SEO best practices
  • Google Analytics
  • Google AdSense
